# HISTORIAS DE USUARIO COMPLETAS

## HU-01 Explorar lugares en mapa
Como usuario, quiero ver lugares en un mapa para descubrir opciones cercanas.

Criterios de aceptacion:
- El mapa carga sin errores.
- Se muestran marcadores de lugares disponibles.
- Al seleccionar un marcador se ve informacion basica del lugar.

## HU-02 Filtrar por categoria
Como usuario, quiero filtrar lugares por categoria para encontrar opciones relevantes.

Criterios de aceptacion:
- Existen al menos 6 categorias visibles.
- El filtro actualiza resultados de forma inmediata.
- Se puede limpiar el filtro.

## HU-03 Registrar reseña
Como usuario autenticado, quiero publicar una reseña para compartir mi experiencia.

Criterios de aceptacion:
- Se permite puntaje de 1 a 5 estrellas.
- Se puede registrar comentario de texto.
- La reseña queda visible en el detalle del lugar.

## HU-04 Buscar por nombre
Como usuario, quiero buscar un lugar por nombre para llegar mas rapido al resultado deseado.

Criterios de aceptacion:
- Busqueda tolerante a mayusculas/minusculas.
- Resultados relevantes segun coincidencia de nombre.
- Mensaje claro cuando no hay resultados.
