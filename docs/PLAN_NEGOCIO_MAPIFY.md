# MAPIFY
## Plan de Negocio — 2026

**Descubre, Comparte y Apoya Negocios Locales**

---

# 1. Identidad Corporativa

## Misión

Empoderar a la comunidad local mediante una plataforma web que facilita el descubrimiento de negocios, genera visibilidad digital para emprendedores y fortalece la confianza mediante reseñas auténticas.

## Visión

Ser la plataforma de referencia para la visibilidad e interconexión de negocios locales en Latinoamérica, destacándose por equidad, transparencia y acceso democrático a oportunidades digitales.

## Valores

- **Equidad:** Visibilidad justa para todos los negocios, independientemente del tamaño.
- **Transparencia:** Reseñas auténticas y feedback confiable.
- **Comunidad:** Fortalecer redes locales y relaciones.
- **Innovación:** Herramientas ágiles y escalables.
- **Sostenibilidad:** Modelo de negocio responsable y duradero.

## Propósito

Democratizar la visibilidad digital de pequeños y medianos negocios, cerrando la brecha entre emprendedores locales y oportunidades de crecimiento en el ecosistema digital.

---

# 2. Problema y Oportunidad

## Problema Identificado

Negocios y lugares locales (restaurantes, tiendas, servicios) enfrentan una **baja visibilidad digital** y carecen de información consolidada, confiable y de fácil acceso. Usuarios buscan lugares cercanos pero disponen de opciones limitadas o fragmentadas.

## Oportunidad de Negocio

Una plataforma integrada que:
- Mejore la visibilidad de negocios locales.
- Centralice información verificable y reseñas de usuarios.
- Genere ingresos mediante destacados, análisis y publicidad segmentada.
- Escale a nivel regional y proporcione datos valiosos para emprendimientos.

# 6. Resumen Ejecutivo

**Mapify** es una plataforma web de código abierto que revoluciona la forma en que comunidades locales descubren negocios y generan visibilidad para emprendedores.

Desarrollada con metodología ágil Scrum (5 sprints × 2 semanas), Mapify entrega un **MVP funcional** que integra:

- Sistema de autenticación por roles (usuarios, administradores)
- Mapa interactivo con geolocalización
- Catálogo CRUD de lugares (negocio, servicios, contacto)
- Sistema de reseñas y calificaciones
- Filtros avanzados por categoría y búsqueda
- Interfaz responsive y accesible

**Visión 2030:** Expandir a nivel regional, monetizar mediante destacados y análisis, y crear un ecosistema de visibilidad digital equitativa.

# 12. Stack Tecnológico

| Capa | Tecnologías | Justificación |
|---|---|---|
| **Frontend** | React 19, Vite, Tailwind CSS, Leaflet, React Router, Axios | Performante, escalable, moderna |
| **Backend** | Node.js, Express.js, JWT | Ligero, rápido, tiempo real |
| **Base de Datos** | MySQL + Prisma ORM | Relacional, transaccional, mantenible |
| **Autenticación** | JWT + roles | Segura, stateless, integrable |
| **Geolocalización** | Leaflet.js + OpenStreetMap | Open source, preciso, sin costo |
| **DevOps** | GitHub, GitHub Actions, CI/CD | Automatización, calidad, velocidad |
| **Hosting** | Render, Vercel (fase 2) | Escalable, sin mantenimiento |

**Decisión clave:** MySQL + Prisma ORM en lugar de NoSQL para garantizar consistencia transaccional en reseñas y relaciones lugar-usuario.

# 16. Análisis FODA

## Fortalezas

- Visión clara y problematica relevante
- Equipo técnico capacitado y comprometido
- Stack moderno y escalable
- Metodología ágil comprobada
- Documentación integral desde inicio

## Oportunidades

- Crecimiento del e-commerce y presencia digital PYME
- Demanda de herramientas accesibles en LATAM
- Potencial de partnerships con ecosistema emprendedor
- Tendencia open source y comunidades tech

## Debilidades

- Recursos limitados (equipo académico)
- Capital inicial reducido en fase 2
- Mercado saturado (competencia: Google Maps, etc.)
- Modelo de ingresos no validado aún

## Amenazas

- Cambios en políticas de privacidad/datos (GDPR, LSRPD)
- Competencia de jugadores grandes
- Economía local impacta en adopción PYME
- Churn de usuarios si UX no es óptima
